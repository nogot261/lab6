## Цель работы

Разработать веб-приложение, которое по маршруту `/size2json/` принимает PNG-изображение в поле `image` формата `multipart/form-data` и возвращает JSON с шириной и высотой изображения.

## Что реализовано

- `GET /login/` возвращает логин Moodle: `dieuvina`.
- `POST /size2json/` принимает PNG-файл из поля `image`.
- Размеры PNG читаются из заголовка файла без сохранения на диск.
- Ответ возвращается в формате JSON: `{"width":123,"height":456}`.
- Для неверного файла или отсутствия поля `image` возвращается ошибка.
- Добавлены CORS-заголовки для `https://kodaktor.ru`.
- Добавлен `Dockerfile` для деплоя на Render.

## Как проверить

После деплоя заменить `<URL>` на адрес Render.

Проверить логин:

```bash
curl <URL>/login/
```

Проверить обработку PNG:

```bash
curl -X POST <URL>/size2json/ \
  -F "image=@test.png;type=image/png"
```

Ожидаемый результат:

```json
{"width":123,"height":456}
```

Значения `width` и `height` должны соответствовать реальному размеру файла `test.png`.
